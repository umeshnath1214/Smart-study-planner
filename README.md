Description
The Smart Study Planner is a simple and user-friendly web-based application designed to help students organize their study schedules efficiently. It allows users to plan tasks, manage study time, and stay productive by keeping track of their daily learning goals.

Features
📝 Add and manage study tasks
⏰ Set study schedules and reminders
📅 Organize daily or weekly plans
✔️ Mark tasks as completed
🎨 Clean and simple user interface
💻 Runs directly in a web browser (no installation required)

How to Open the Project
Download or clone the project files to your computer
Locate the file named Smart study.html
Right-click on the file
Click Open with → Any Web Browser (Chrome, Edge, etc.)
The Smart Study Planner will open and be ready to use
